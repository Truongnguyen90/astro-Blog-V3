═══════════════════════════════════════════════════════════════════
TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

If you get "provider is not enabled" error:
→ GitHub provider not enabled in Supabase
→ Go back to authentication providers and enable it

If callback fails:
→ Check callback URL matches exactly
→ Clear browser cookies and try again

If you see "Invalid API key":
→ Get correct anon key from Supabase Settings > API
→ Update .env file
→ Restart dev server
